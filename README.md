# Tableau Assignments

## Overview

This repository contains a collection of Tableau assignments completed as part of learning and practicing Business Intelligence (BI), Data Visualization, Dashboard Development, and Data Analysis using Tableau.

The assignments cover a wide range of Tableau concepts including data connections, data preparation, calculated fields, parameters, sets, groups, joins, blending, forecasting, statistical analysis, dashboard design, KPI creation, and advanced visualizations.
